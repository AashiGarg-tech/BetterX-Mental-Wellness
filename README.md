# BetterX – Mental Wellness Platform

BetterX is a student mental wellness platform built to support students through counselling, mood tracking, chatbot support, resources, and community interaction.

## 🚀 Run the project locally

### 1. Start backend server

Open terminal and run:

```bash
cd minddorm-mental-wellness
cd backend
node server.js
```

Keep this terminal running.

---

### 2. Start frontend

Open a **new terminal** and run:

```bash
cd minddorm-mental-wellness
npm i
npm start
```

Frontend will open on:

```bash
http://localhost:3000
```

Backend runs on:

```bash
http://localhost:5050
```

---

## ✨ Features

- Student authentication

- Mental wellness chatbot

---

## ⚠️ Important Notes

- Start backend before frontend.
- Keep backend terminal running while using the app.
- Make sure `.env` is configured properly.
- MongoDB and PostgreSQL should be connected before running.
